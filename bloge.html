Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. They are written below :
1. getElementById: When a specific element is needed, it is used by taking the I'd name of that element. Due to which that element can be accessed very easily.
2. getElementsByClassName: When you want to get some elements together, it is used by giving the same Class name of each element.
3. getElementsByTagName: When you need all the elements with the same Tag, it is used.
4. quarySelectorAll: This is also used when you need all the elements with the same class name.
5.quarySelector: When you need to get that element with its class name, it is used.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

These three properties are used to get or set the inner content of an element in the DOM (Document Object Model). However, they work differently.
1. innerHTML : It gets or sets the full HTML code (including tags) of an element and it parses the HTML and renders the inner tags. It is possible to inject new HTML tags through it.
2. innerText : When we are needs inner text in a tags then we are use the innerText property. By using innerText we can also set the internal value of a tag.
3. textContent : This returns the full text content of the element, even if hidden by CSS. Ignores HTML tags and shows the full text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique of javascript, where an event listener is set on a parent element and its child element's event is handled.
This is especially useful when adding event handlers for multiple child elements of the same type.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a behavior of JavaScript DOM Events, This is a process of event handling where an event first triggers a child element, then peopagates step by step to the parent elements above.
And it can be stopped by stopPropagate and stopImiditePropagate.
